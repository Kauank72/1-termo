# Resumo das Disciplinas: Arquivo Geral de Aulas

## 1. Arquitetura IoT
*   **Foco:** Integração entre Hardware e Software.
*   **Tecnologias:** Arduino (Camada de Percepção), C++ (Firmware/Controle) e Python (Gateway/Processamento de Dados).
*   **Conceito Chave:** Fluxo de dados do sensor -> Arduino -> Comunicação Serial -> Aplicação Python.

## 2. Levantamento de Requisitos
*   **Foco:** Definição do escopo do projeto de software.
*   **Requisitos Funcionais (RF):** O que o sistema faz (ex: cadastrar usuário, gerar relatório).
*   **Requisitos Não Funcionais (RNF):** Como o sistema se comporta (ex: segurança, velocidade, disponibilidade).
*   **Técnicas:** Entrevistas, brainstorming e prototipagem.

## 3. Lógica de Programação
*   **Foco:** Fundamentos de algoritmos comparando C++ e Python.
*   **Variáveis:** Tipagem forte (C++) vs Tipagem dinâmica (Python).
*   **Estruturas:** Condicionais (If/Else) e Repetição (Loops For/While).
*   **Diferença:** C++ focado em performance/sintaxe rígida; Python focado em produtividade/legibilidade.

## 4. Sistemas Operacionais
*   **Foco:** Infraestrutura, Virtualização e Segurança.
*   **Ambientes:** Distribuições Linux (Debian, Red Hat, Kali) e Windows CLI (CMD/PowerShell).
*   **Virtualização:** Uso de Hypervisors (VirtualBox) para isolamento de sistemas.
*   **Segurança:** Gestão de privilégios, Firewalls e análise de vulnerabilidades.
